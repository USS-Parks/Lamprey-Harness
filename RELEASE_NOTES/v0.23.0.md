# Lamprey v0.23.0 — Source publication notes

This source cut adds conversation-bound PR Chat, approval-gated reviews, editable patch
proposals, inline visualizations, and direct artifact editing. It is published on `main`
without a v0.23.0 tag or binary release.

## In plain language

Lamprey can now stay in one conversation while you inspect a pull request, talk through a
specific hunk, and decide what to post or apply. It does not publish comments or touch files
behind your back: those steps remain visible, targeted, and approval-gated.

## What changed

- Bind a GitHub pull request to a conversation with immutable repository ID and base/head SHA.
- Inspect bounded PR summary, files, diff hunks, checks, review comments, and patch context.
- Draft pending reviews, inline annotations, replies, and final review submissions through
  exact-target approval and idempotency receipts.
- Record detached findings locally without posting to GitHub.
- Propose and edit patches without touching files; explicitly accept through Lamprey's normal
  workspace patch authority or reject without a workspace change.
- Fail closed on stale heads and path escapes; restore all affected files when a later hunk fails.
- Extend the existing PR panel with Chat about this PR, selected-hunk send, check progress,
  annotations, review confirmation, and editable patch cards.

## Verification boundary

PR-1 through PR-5 passed their focused type, lint, unit/native, build, and renderer gates.
PR-6 records the full repository gate in `PLANNING/CJ26_PR_CHAT_AFTER.md`. The disposable
GitHub owner playbook remains open, so v0.23.0 is source implementation-complete only.
Packaged downloads intentionally remain on v0.21.0 until separately authorized.
