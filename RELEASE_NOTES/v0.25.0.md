# Lamprey v0.25.0 — Browser Developer Mode

**Release status:** prepared locally; not published

Browser Developer Mode adds explicit, OFF-by-default CDP inspection to the existing Browser
panel:

- bounded, redacted console and network observations with opt-in response bodies;
- structured DOM/accessibility, fixed runtime, layout/performance, trace, and screenshot tools;
- exact-origin Ask/Allow/Deny trust and shared approval metadata;
- Windows PowerShell AST danger inspection before shell approval;
- visible target, recording counts, annotated evidence, clear, and detach controls.

The feature remains off until the user enables it and explicitly allows an exact HTTP(S)
origin. The owner packaged-app playbook remains open, so these notes describe the verified
local implementation and do not claim completed live Codex parity.
