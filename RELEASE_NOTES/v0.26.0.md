# Lamprey v0.26.0 — Automations and Operational Goals

**Release status:** v0.26.0 publication authorized 2026-07-19

Lamprey's local automations and goals now form one governed control plane:

- strict model tools list, create/update, delete, and run automations;
- typed one-shot reminders, schedules, events, and monitors with durable dedup/retry state;
- operational goal lifecycle with actor provenance, token/time budgets, blockers, and
  completion evidence;
- one goal-owned bounded loop that automations can wake through the existing controller;
- shared Automations management and Plans & goals lifecycle/progress controls.

Recurring autonomy stays OFF by default. Global ceilings remain authoritative, and goal or
automation caps can only tighten them. The owner packaged-app background/restart playbook is
still open, so these notes describe the verified local implementation and do not claim
completed live Codex parity.
